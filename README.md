 📊 Personal Finance Analytics Dashboard

 📌 Project Overview

This project is a Personal Finance Analytics Dashboard created using Microsoft Excel. The dashboard analyzes monthly expenses, spending categories, and payment methods to provide meaningful financial insights.

 🎯 Objectives

* Track monthly expenses
* Analyze category-wise spending
* Visualize expense distribution
* Identify major spending areas
* Build an interactive dashboard

 🛠️ Tools Used

* Microsoft Excel
* Charts & Visualizations
* SUMIF, MAX, MIN, COUNTA formulas
* Conditional Formatting

 📊 Dashboard Features

* Total Expense Analysis
* Highest & Lowest Expense Tracking
* Category-wise Expense Distribution
* Pie Chart Visualization
* Bar Chart Analysis
* Spending Insights

 📁 Files Included

* Personal_Expense_Analytics.xlsx
* dashboard.png

 🔍 Key Insights

* Shopping and Rent contributed the highest expenses.
* Transportation expenses were comparatively lower.
* Multiple payment methods were analyzed for spending behavior.

🚀 Outcome

This project helped in understanding data analysis, dashboard creation, visualization techniques, and financial tracking using Excel.


Created as part of a Data Analytics Internship Project.
